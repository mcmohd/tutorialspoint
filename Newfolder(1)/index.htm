WCF Tutorial


WCF stands for Windows Communication Foundation and is a framework for building, configuring and deploying network-distributed services. Earlier known as Indigo, it enables hosting services in any type of operating system process.

The tutorial is intended to explain the fundamentals of WCF and is conveniently divided into various sections to gradually move to an advanced level. Every section of this tutorial has adequate number of examples for an ease in understanding of different concepts of WCF.

Audience

The objective of this tutorial is to benefit the absolute beginners who want to learn WCF. After completing this tutorial, beginners will acquire an understanding of WCF to have the basic groundwork necessary for a higher level of expertise.

Prerequisites

Before proceeding with this tutorial, you should have basic knowledge of .NET framework.
